Sidebar Template Demo

This page demonstrates the sidebar template functionality in the default theme. The sidebar will appear on the right side of the content on desktop devices and above the content on mobile devices.

Features

  • Responsive Layout: Sidebar stacks above content on mobile devices
  • Widget Support: Displays widgets from the left-sidebar configuration
  • Navigation: Shows sidebar menu items if configured
  • Related Pages: Displays child pages if they exist
  • Sticky Positioning: Sidebar stays in view while scrolling on desktop

How to Use

To use this template:

  1. Select "Sidebar Demo" as your page template in the admin interface
  2. Configure widgets in the left-sidebar section
  3. Add a sidebar menu if desired
  4. The template will automatically handle the layout and responsiveness

Customization

You can customize the sidebar by:

  • Modifying the sidebar.html.mod file to change sidebar content
  • Adjusting CSS variables in style.css for colors and spacing
  • Adding new widget types to the sidebar module
  • Creating additional sidebar variations for different page types
