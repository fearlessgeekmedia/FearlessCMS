Custom admin path, Whisperwind theme, Pay what you want web design/development


We have added an important security feature to the CMS. The admin path is now customizable. This is a big deal. It means that the admin path can be changed to something other than /admin. This is important because it makes it more difficult for hackers to find the admin path.

We have also added a new theme, Whisperwind. It's a simple, clean, and modern theme that is easy on the eyes. It's donloadable at Github.
