Parallax Plugin - New Features Test

This page demonstrates the new CSS class and custom ID features added to the parallax shortcode.

1. CSS Class Support

The new class attribute allows you to add custom CSS classes to parallax sections:

[parallax_section id="hero-section" background_image="/uploads/hero-bg.jpg" speed="0.3" effect="scroll" class="hero-parallax dark-theme custom-spacing"]

Welcome to Our Website

Experience stunning parallax effects

[/parallax_section]
Benefits:
  • Add multiple CSS classes separated by spaces
  • Customize styling for specific sections
  • Create theme variations (dark-theme, light-theme, etc.)
  • Apply responsive design classes

2. Custom ID Support

The new custom_id attribute provides an additional identifier for JavaScript and CSS targeting:

[parallax_section id="contact-section" background_image="/uploads/contact-bg.jpg" speed="0.6" effect="fixed" class="contact-section" custom_id="main-contact-form"]

Get In Touch

Ready to start your next project?

[/parallax_section]
Benefits:
  • Separate from the required id attribute
  • Stored as a data-custom-id attribute
  • Perfect for JavaScript event handling
  • CSS targeting without conflicting with existing IDs

3. Combined Usage Example

You can use both features together for maximum flexibility:

[parallax_section id="services" background_image="/uploads/services-bg.jpg" speed="0.4" effect="fade-in" class="services-section light-overlay premium-theme" custom_id="main-services-grid"]

Our Services

Web Design
Development
Consulting
[/parallax_section]

4. CSS Styling Examples

Here's how you can use the custom classes in your CSS:

/* Custom styling for hero parallax sections */ .hero-parallax { min-height: 600px; } /* Dark theme styling */ .dark-theme .parallax-content { background: rgba(0,0,0,0.7); } /* Custom spacing for specific sections */ .custom-spacing .parallax-content { padding: 4rem 2rem; } /* Premium theme styling */ .premium-theme .parallax-content { background: linear-gradient(135deg, rgba(255,215,0,0.1), rgba(255,140,0,0.1)); }

5. JavaScript Usage Examples

Target parallax sections using the new attributes:

// Target by custom ID const heroSection = document.querySelector('[data-custom-id="main-hero-section"]'); // Target by custom class const heroSections = document.querySelectorAll('.hero-parallax'); // Add custom event listeners heroSection.addEventListener('click', function() { console.log('Hero section clicked!'); }); // Apply custom animations to premium themes const premiumSections = document.querySelectorAll('.premium-theme'); premiumSections.forEach(section => { section.style.border = '2px solid gold'; });
Important Notes:
  • The id attribute is still required and must be unique
  • The class attribute supports multiple classes separated by spaces
  • The custom_id attribute is stored as data-custom-id in the HTML
  • Custom classes are combined with the default generated classes
  • All existing parallax functionality remains unchanged

6. Migration Guide

Existing parallax shortcodes will continue to work without modification. To add the new features:

// Before (still works) [parallax_section id="section1" background_image="/image.jpg" speed="0.5"] Content here [/parallax_section] // After (with new features) [parallax_section id="section1" background_image="/image.jpg" speed="0.5" class="my-custom-class" custom_id="unique-identifier"] Content here [/parallax_section]
